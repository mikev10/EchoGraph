# EchoGraph CLI

Context Engineering CLI for Claude Code.

## Installation

```bash
pip install echograph
```

## Quick Start

```bash
# Initialize Context Engineering in your project
echograph init

# Check your setup
echograph doctor

# Validate context files
echograph validate
```

## Commands

- `echograph init` - Scaffold `.claude/` directory with templates
- `echograph update` - Update templates preserving customizations
- `echograph validate` - Check context files for completeness
- `echograph doctor` - Verify Claude Code setup
